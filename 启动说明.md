# 项目启动说明

## 方式一: 一键启动 (推荐)

使用 `start.py` 脚本一键启动前端和后端:

```bash
# 激活 conda 环境
conda activate yolo
cd E:\bysj\ultralytics-8.2.27\project
python start.py
```

**功能特性**:
- ✅ 自动检查系统依赖 (Conda, Node.js)
- ✅ 自动检查并提示安装前端依赖
- ✅ 同时启动前端和后端服务
- ✅ 彩色终端输出,状态一目了然
- ✅ Ctrl+C 一键关闭所有服务
- ✅ 自动监控进程状态

**访问地址**:
- 前端: http://localhost:5173
- 后端: http://localhost:8000
- API 文档: http://localhost:8000/docs

---

## 方式二: 手动启动

### 1. 启动后端

```bash
cd E:\bysj\ultralytics-8.2.27\project
conda activate yolo
uvicorn main:app --reload --port 8000
```

### 2. 启动前端

打开新终端:

```bash
cd E:\bysj\ultralytics-8.2.27\project\frontend
npm run dev
```

---

## 环境要求

- **Python**: 3.8+
- **Conda**: Anaconda 或 Miniconda
- **Node.js**: 16+
- **Conda 环境**: yolo (需提前创建并安装依赖)
