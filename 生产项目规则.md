## 3. 数据库规范

### 3.1 表设计规范

#### 命名规范
- 表名：小写字母，下划线分隔，复数形式（如 `detection_results`）
- 字段名：小写字母，下划线分隔（如 `created_at`）
- 主键：统一使用 `id`
- 外键：`关联表_id`（如 `user_id`）

#### 字段规范
```sql
-- ✅ 正确示例
CREATE TABLE detection_results (
    id INT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    image_path VARCHAR(512) NOT NULL COMMENT '图片路径',
    user_id INT COMMENT '用户ID',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    INDEX idx_user_id (user_id),
    INDEX idx_created_at (created_at),
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='检测结果表';

-- ❌ 错误示例
CREATE TABLE DetectionResults (  -- 表名不规范
    ID int,  -- 字段名不规范
    ImagePath text,  -- 类型不合适
    CreateTime datetime  -- 命名不统一
);
```

### 3.2 索引规范

- 主键自动创建索引
- 外键字段创建索引
- 频繁查询字段创建索引
- 避免过多索引（影响写入性能）

```sql
-- 单列索引
CREATE INDEX idx_username ON users(username);

-- 复合索引
CREATE INDEX idx_user_created ON detection_results(user_id, created_at);

-- 唯一索引
CREATE UNIQUE INDEX uk_username ON users(username);
```

### 3.3 数据迁移规范

#### 使用迁移脚本
```python
# update_database.py
def migrate_add_security_questions():
    """添加安全问题字段"""
    try:
        # 1. 检查字段是否存在
        cursor.execute("SHOW COLUMNS FROM users LIKE 'security_question'")
        if cursor.fetchone():
            print("字段已存在，跳过")
            return
        
        # 2. 添加字段
        cursor.execute("""
            ALTER TABLE users 
            ADD COLUMN security_question VARCHAR(255),
            ADD COLUMN security_answer VARCHAR(255)
        """)
        
        # 3. 提交事务
        conn.commit()
        print("迁移成功")
    except Exception as e:
        conn.rollback()
        print(f"迁移失败: {e}")
```

#### 迁移原则
- 每次迁移创建独立脚本
- 迁移可回滚
- 记录迁移历史
- 生产环境先备份

## 5. 安全规范

### 5.1 密码安全

```python
# ✅ 正确：使用哈希加密
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def hash_password(password: str) -> str:
    return pwd_context.hash(password)

def verify_password(plain_password: str, hashed_password: str) -> bool:
    return pwd_context.verify(plain_password, hashed_password)

# ❌ 错误：明文存储
password = "123456"  # 绝对禁止
```

### B. 配置文件模板

```python
# config.py
from pydantic import BaseSettings

class Settings(BaseSettings):
    # 数据库配置
    DATABASE_URL: str
    
    # 模型配置
    MODEL_PATH: str = "weights/yolov8n.pt"
    
    # 算法参数
    CONF_THRESHOLD: float = 0.25
    IOU_THRESHOLD: float = 0.45
    GAP_THRESHOLD: float = 0.5
    ROW_THRESHOLD: float = 0.6
    
    # 安全配置
    SECRET_KEY: str
    ALGORITHM: str = "HS256"
    ACCESS_TOKEN_EXPIRE_MINUTES: int = 30
    
    class Config:
        env_file = ".env"

settings = Settings()
```
